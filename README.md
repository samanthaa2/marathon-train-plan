# marathon-train-plan

## Project Overview
For my final project, I created a marathon training planner that builds a personalized training schedule and then adjusts it based on the user’s progress. The idea came from one of my dream goals, which is to run a marathon. I wanted to make something that connects programming to a real goal I care about.

This program starts by asking the user for information like their current weekly mileage, how many days per week they can run, and how many weeks they have until race day. Based on that, it creates a training plan. After each workout, the user can enter stats like distance, time, and how hard the run felt. At the end of each week, the program uses that information to make changes to the next week’s plan if needed.
